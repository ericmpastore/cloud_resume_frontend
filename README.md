# cloud_resume
With cloud platforms like Google Cloud Platform (GCP) an essential part of data engineering, this project will demonstrate how to host a static resume as a webpage on GCP.

## Project Log:
- Developed static resume page **index.html**
- Developed css file **cloud.css**
- Developed javascript file **counter.js**
- Launched Cloud Storage bucket on Google Cloud Platform
- Added security roles to storage bucket
- Loaded project files to storage bucket
- Granted access to allUsers allowing index.html to be visible at https://storage.googleapis.com/cloud-resume-pastore/index.html
